Documentation

Presented documentation holds the information about URL endpoints, reguests and responses.

Business Logic URL HTTP Method Reguest Response
Return a list of ALL movies to the user /movies GET None A json object holding data about movies
Return data about a single movie by title to the user /movies/:title GET None A JSON object holding data about selected movie, containing title, genre, description, etc. 
                            {
                                    title: String, 
                                    genre: String, 
                                    description: String, 
                                    director: {name: String,
                                               bio: String 
                                    },
                                    actors: [String],
                                    imageUrl: String,
                                    featured: boolean
                                }
Example: 
                            {
                                    title: 'The Lord of the Rings', 
                                    genre: 'Fantasy', 
                                    description: String, 
                                    director: {name:'Peter Jackson',
                                               bio: 'Sir Peter Robert 
                                                 Jackson ONZ KNZM is a 
                                                 New Zealand film director,
                                                 screenwriter, and film 
                                                 producer. He is best known
                                                 as the director, writer, 
                                                 and producer of the Lord 
                                                 of the Rings trilogy and 
                                                 the Hobbit trilogy, both 
                                                 of which are adapted from 
                                                 the novels of the same name 
                                                 by J. R. R. Tolkien.' 
                                    },
                                    actors: 'Elijah Wood','Viggo Mortensen',
                                    'Sean Astin','Orlando Bloom', etc.
                                    imageUrl: String,
                                    featured: true
                                }
Return data about a genre by title /movies/genre/:name GET None A JSON object holding data about selected movie, containing title, genre, description, etc. 
                            {
                                    genre: String, 
                                    description: String 
                                    }
Example: 
                            {
                                    genre: 'Drama', 
                                    description: String 
                                    }
Return data about a director by name /movies/director:name GET None A JSON object holding data about selected director. 
                            {
                                    name: String, 
                                    bio: String, 
                                    birth: Date, 
                                    death: Date
                                 }
Example: 
                            {
                                    name: 'James Cameron', 
                                    bio: 'James Francis Cameron CC is a Canadian filmmaker.', 
                                    birth: 1954/08/16, 
                                    death: ---
                                 }
Allow new users to register /users POST A JSON object holding data about the user to register. 
                            
                                { 
                                    username: String, 
                                    favoriteMovies: []
                                }
Example: 
                                
                                    { 
                                        username: 'John', 
                                        favoriteMovies: []
                                    }
A JSON object holding data about the user that was registered. 
                            
                               {
                                   username: String, 
                                   id: Number
                                   favoriteMovies: [
                                   ...
                                   ]
                               }
Example: 
                            
                               {
                                   username: 'John', 
                                   id: 1
                                   favoriteMovies: [
                                   ...
                                   ]
                               }
Allow users to update their user info (username) /users/:username PUT None A text message indicating thatthe user info (userinfo) was updated
Allow users to add a movie to their list of favorites /users/:id/:title POST A JSON object holding data about movie to add to list of favorites. Example: 
                        
                            {
                                id: Number
                            }
A JSON object holding data about the movie that was added to list of favorites
Allow users to remove a movie from their list of favorites /users/username/:movie DELETE None A text message that a movie was deleted from favorite list
Allow existing users to deregister /users/:username DELETE None A text notification that user deregisterd
